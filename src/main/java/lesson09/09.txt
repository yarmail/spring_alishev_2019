Урок 9: Аннотации. Введение.

Java аннотации - это специальный тип комментариев,
с помощью которого можно:
- передавать инструкции для Java компилятора
(например @Override)
- передавать какие-либо инструкции для анализаторов
исходного кода
- передавать метаданные, которые
могут быть испольхованы либо вашим Java приложением
(с помощью рефлексии), либо другими приложениями
или фреймворками (например Spring)

---

Зачем использовать аннотации?
короче, удобнее, читабельнее

---

Конфигурация с помощью анотаций
Мы помечаем классы, которые хотим сделать бинами
специоальной анотацией @Component и потом с помощью
специальных инструкций сканируем папку с такими классами,
после чего ими можно пользоваться как бинами

Рядом с анотацией @Component можно указать
id создаваемого бина
@Component ("nameOfBean")
Если не указывать - тогда название бина будет -
"название_класса_с_маленькой_буквы"

---
Сначала меняем конфиг файл, прописываем там
папку, которую надо сканировать на бины
